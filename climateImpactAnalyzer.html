Climate Change Impact Analyzer

Analysis and predictive data

This project focused on analyzing hurricane data from 1900 to the present to explore trends, cluster storm impacts, and detect anomalies. The dataset was cleaned to include only hurricane-specific attributes—magnitude, start year, and adjusted damage costs in USD—before being used in predictive models and visualizations. Click the GitHub link below to see the full repository.

GitHub Repository

My primary contributions centered on developing the visualization system and implementing the machine learning algorithms. Using Matplotlib and mplcursors, I designed interactive visualizations that allow users to explore hurricane data through line graphs (windspeed trends), scatter plots (clustered storm impacts), and bar charts (highlighting anomalies). On the algorithmic side, I implemented:

Linear Regression to predict windspeed trends for the next 15 years.

KMeans Clustering to group hurricanes by severity and economic damage.

Local Outlier Factor (LOF) to detect rare hurricanes with unusual windspeed–damage relationships.

Through these tools, I was able to surface patterns in hurricane intensity and damage, highlight meaningful clusters, and detect rare outliers that deviate from expected behavior. The project combines data science, predictive modeling, and interactive visualization to make complex hurricane data more interpretable and actionable.
