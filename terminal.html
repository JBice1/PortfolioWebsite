Comprehensive Shell

A user-friendly unix shell

This is a comprehensive shell that enhances process control, user interaction, and error handling. This shell uses parent-child relations, process creation, and user-input parsing and verification to manage and execute user commands. It allows for the path search specified in the $PATH environment varibale, ensuringing the shell can execute commands regardless of the current working directory. This shell also supports essential shell features such as input/output redirection, piping, and background processing. All of these features combine to make a robust and user freindly shell interface that allows the user to effieciently interact with the operating system.
